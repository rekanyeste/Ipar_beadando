# Ipar 4.0 Beadandó Real-Time Face recognition
## Készítette: Nyeste Réka (GKE37T)
### Forrás: https://towardsdatascience.com/real-time-face-recognition-an-end-to-end-project-b738bb0f7348

# Jelenleg két python file található a programban, amiket főleg eszközein teszteléséhez készítetem
## Első: test.py -> ezzel ellenőriztem, hogy működik-e az importált opencv verziója. A python nyelvnél nagyon fontos, hogy a megfelelő verziójú csomagokat és programokat teleptítsük.
## Második: cameraTesting.py -> a számítógép vagy laptop kamerájának a tesztelésére elkészített program. Érdemes lefuttatni, hogy minden rendben van-e mielőtt hozzálátunk a többi feladathoz. Csak simán futtassuk le a programot, majd ha látjuk, hogy működik a kamera 'ESC' billentyűvel kiléphetünk belőle.
## Harmadik: faceDetection.py -> Az arcunkat ismeri fel a kamerában. Egyelőre csak felismeri azt és nem társítja névhez, pl.: nem tudja megkülönböztetni egy híresség arcát a sajátomtól. Csak felismeri, hogy a kamerában egy emberi arc van
## Negyedik: eyeDetection.py -> ugyanaz, mint a harmadik program, csak itt az emberi szemeket ismeri fel (meg néha az orromat...)
### A faceDetection program tartalmazza az eyeDetection funkciót is!
### Az eyeglasses "kaszkád" sokkal jobban felismeri a szemeket szemüvegben, mint a sima eyeCascade, ezért én ezt preferálom.
## Ötödik: FaceRecognition01.py -> Elment 5 db szürkeárnyalatos képet a kiválasztott mappába, miután inputba írtunk egy ID-t és detektálta a kamerában lévő arcot. Ez a program egy lépése annak, hogy a saját arcunkat ismerje fel a végén a programunk.
## Hatodik: FaceRecognition02.py -> A legfontosabb rész. Ez a program tanulja meg az előző programból elmentett képek alapján felismerni az adott személynek az arcát. Mikor lefutott egy yaml fájlt készít a trainer mappába, ez tartalmazza azokat az adatokat, amik a következő lépéshez, vagyis egy arc megismeréséhez szükségesek. Ergo, ha belenézünk a kamerába felismer.
